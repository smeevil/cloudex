    Modules

    Cloudex

    Cloudex takes care of uploading image files or urls to Cloudinary

    Cloudex.CloudinaryApi

    Defines behaviour of the API implementation of Cloudinary

    Cloudex.CloudinaryApi.Live

    The live API implementation for Cloudinary uploading

    Cloudex.CloudinaryApi.Test

    A simple stub of cloudinary api responses to speed up testing

    Cloudex.EnvOptions

    A simple module to get System Environment Variables (ENV VARS)

    Cloudex.Settings

    This is a GenServer that contains the API settings for cloudinary You can interact with the get function to retreive the settings you want

    Cloudex.UploadedImage

    A simple struct containing all the information from cloudinary

    Cloudex.Url

    A module that helps creating urls to your cloudinary image, which can optionally transform the image as well
